Booleans

Let's specify two rewriting rules.

  • NOT FALSE → TRUE
  • NOT TRUE → FALSE

Will the following statement evaluate to TRUE or FALSE?

  • NOT NOT NOT FALSE

We can find out by rewriting according to our rewriting rules.

  • NOT NOT NOT FALSE
  • NOT NOT TRUE
  • NOT NOT TRUE
  • NOT FALSE
  • NOT FALSE
  • TRUE

Because NOT NOT NOT FALSE can be rewritten to TRUE, and TRUE cannot be rewritten, we say that TRUE is a normal form of NOT NOT NOT FALSE. TRUE and FALSE are the only normal forms possible with these rules.

Numbers

Let's specify rewriting rules for natural numbers.

  • NUMBER → ZERO
  • NUMBER → SUCCESSOR NUMBER

Let's try to rewrite NUMBER to express the number two.

  • NUMBER
  • SUCCESSOR NUMBER
  • SUCCESSOR NUMBER
  • SUCCESSOR SUCCESSOR NUMBER
  • SUCCESSOR SUCCESSOR NUMBER
  • SUCCESSOR SUCCESSOR ZERO

We can see that SUCCESSOR SUCCESSOR ZERO is a normal form of NUMBER, and that NUMBER has an endless number of possible normal forms. No normal forms of NUMBER contain NUMBER, since NUMBER can always be rewritten.

Normal Forms

Let's specify rewriting rules for determining the evenness of a number.

  • EVENNESS ZERO → TRUE
  • EVENNESS SUCCESSOR x → NOT EVENNESS x | x : NUMBER

The second rule states that for anything that is a normal form of NUMBER, if call that thing x, that EVENNESS SUCCESSOR x can be rewritten to NOT EVENNESS x.

Let's use all of these rules to determine if the number three is even.

  • EVENNESS SUCCESSOR SUCCESSOR SUCCESSOR ZERO
  • EVENNESS SUCCESSOR SUCCESSOR SUCCESSOR ZERO
  • NOT EVENNESS SUCCESSOR SUCCESSOR ZERO
  • NOT EVENNESS SUCCESSOR SUCCESSOR ZERO
  • NOT NOT EVENNESS SUCCESSOR ZERO
  • NOT NOT EVENNESS SUCCESSOR ZERO
  • NOT NOT NOT EVENNESS ZERO
  • NOT NOT NOT EVENNESS ZERO
  • NOT NOT NOT TRUE
  • NOT NOT NOT TRUE
  • NOT NOT FALSE
  • NOT NOT FALSE
  • NOT TRUE
  • NOT TRUE
  • FALSE

Arithmatic

Here we will specify some rules for arithmatic using polish notation.

  • PLUS x ZERO → x | x : NUMBER
  • PLUS x SUCCESSOR y → SUCCESSOR PLUS x y | x, y : NUMBER

The following statement is a representation of the addition of two and two.

  • PLUS SUCCESSOR SUCCESSOR ZERO SUCCESSOR SUCCESSOR ZERO

Let's show that four is a normal form of two plus two.

  • PLUS SUCCESSOR SUCCESSOR ZERO SUCCESSOR SUCCESSOR ZERO
  • SUCCESSOR PLUS SUCCESSOR SUCCESSOR ZERO SUCCESSOR ZERO
  • SUCCESSOR PLUS SUCCESSOR SUCCESSOR ZERO SUCCESSOR ZERO
  • SUCCESSOR SUCCESSOR PLUS SUCCESSOR SUCCESSOR ZERO ZERO
  • SUCCESSOR SUCCESSOR PLUS SUCCESSOR SUCCESSOR ZERO ZERO
  • SUCCESSOR SUCCESSOR SUCCESSOR SUCCESSOR ZERO
